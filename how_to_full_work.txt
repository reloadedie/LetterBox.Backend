Как запустить проект "LetterBox"?

1. Перед тем, как запускать, у вас должно быть установлено:
PostgreSQL
React
C# .net (9.0)

2. выполните действия в read_me_backend.txt

3. выполните действия в read_me_frontend.txt

4. если будете запускать на нескольких компьютерах, нужно находиться в одной подсети (один тип подлючения - проводной/беспроводной)
также разрешите доступ к компьютеру по сети и поставьте галочки в частных сетях необходимые
пропишите в файле фронта.env ip адрес и порт бекенда, во фронте в файле src/providers/ApiProvider.tsx проконтролируйте что приходит
в бекенде ip в Program в API проекте
